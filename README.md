# Hi there 👋,

I'm Ali, a Full Stack AI Engineer working mostly with Python and React.

On the backend, I build REST APIs with Django (DRF) and FastAPI, backed by PostgreSQL, with Redis for caching and Celery for background jobs.

On the AI side, I build agentic and voice-based systems — using LangChain, LangGraph, and OpenAI APIs, with Deepgram and ElevenLabs for real-time voice. Mostly things like booking agents, voice commerce flows, and RAG pipelines that need to actually work in production, not just in a demo.

On the frontend, I work with React and Next.js to wire everything together end to end.

For deployment, I work with AWS and Docker, and Git keeps everything sane along the way.

Outside of engineering, I'm probably playing table tennis or reading a book.
